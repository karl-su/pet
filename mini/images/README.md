# 图片资源说明

请将以下图片文件放置在此目录下：

## Tab图标
- `map.png` - 地图页面未选中图标
- `map-active.png` - 地图页面选中图标  
- `pet.png` - 宠物页面未选中图标
- `pet-active.png` - 宠物页面选中图标

## 地图标记
- `marker.png` - 地图标记点图标

## 宠物头像
- `dog-avatar.png` - 默认狗狗头像
- `cat-avatar.png` - 默认猫咪头像
- `default-pet.png` - 默认宠物头像

## 图标规格建议
- Tab图标：81px × 81px
- 地图标记：30px × 30px  
- 宠物头像：120px × 120px
