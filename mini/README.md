# 宠物小程序

一个基于微信小程序框架的宠物管理应用，包含地图导航和宠物管理功能。

## 功能特性

### 🗺️ 宠物站（地图页面）
- 显示当前位置地图
- 标记附近宠物店位置
- 宠物店详细信息展示
- 一键拨号和导航功能
- 宠物店评分和服务展示

### 🐾 我的宠物（宠物管理页面）
- 宠物列表展示
- 添加新宠物信息
- 编辑宠物资料
- 删除宠物记录
- 宠物健康状态管理

## 技术栈

- 微信小程序原生框架
- 地图API集成
- 本地存储管理
- 响应式UI设计

## 开发说明

1. 使用微信开发者工具打开项目
2. 配置小程序AppID
3. 添加必要的图片资源到images目录
4. 在微信开发者工具中预览和调试

## 权限配置

小程序需要以下权限：
- 位置权限：用于获取用户当前位置
- 相册权限：用于选择宠物头像
- 相机权限：用于拍摄宠物照片

## 注意事项

- 地图功能需要配置腾讯地图API
- 图片资源需要自行添加
- 建议在真机上测试位置相关功能
